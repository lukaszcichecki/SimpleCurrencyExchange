# SimpleCurrencyExchange
Simple Currency Exchange - Rest Api 
using Spring Boot 3, JPA, H2 Database (in-memory), NBP Web Api

## Explore Rest APIs

The app defines following APIs.

    GET /api/account/{id}
    GET /api/account/{id}/{code}
    
    POST /api/account
    
    PUT /api/account/{id}/exchange/{code}
